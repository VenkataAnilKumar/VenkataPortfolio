# API Specification

## User APIs
- POST /signup
- POST /login
- GET /profile

## Course APIs
- GET /courses
- POST /courses
- GET /courses/{id}
- POST /courses/{id}/enroll

## Quiz APIs
- GET /courses/{id}/quizzes
- POST /courses/{id}/quizzes/submit

## Admin APIs
- GET /admin/users
- DELETE /admin/users/{id}
